# <a name="kubernetes-in-the-cloud-adoption-framework"></a>在雲端採用架構中的 Kubernetes

請參閱包含工具、程式和內容 (最佳作法、設定範本和架構指引) 的規範性架構，以簡化大規模採用 Kubernetes 和雲端原生的做法。

必要動作清單會依角色分類，以在 Kubernetes 上成功部署應用程式，從概念證明到生產環境，然後調整和最佳化。
