# <a name="cloud-adoption-antipatterns"></a>雲端採用反模式

客戶通常會遇到雲端採用 [反模式](../organize/fiefdoms-silos.md)。 在遷移至雲端時，這些過失通常會在設計、規劃或執行時發生。 反模式可以封鎖創新，並防止企業採用和實現目標。

下表列出這些模式發生的反模式和方法（或雲端採用階段）。 連結的文章提供每個反模式和方案的範例。
